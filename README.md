# Linux

Linux commands are used in daily work. You can use it in your project.

## Run Locally

Create a bash script:

```bash
  touch script.sh
```

Edit via the example editor, paste your needed commands, save, and exit from the file:

```bash
  vim script.sh
```

Make the file executable:

```bash
  chmod +x script.sh
```

Run the script:

```bash
  ./script.sh
```
